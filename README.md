# Querator_Cam
使用UDIC lab所開發的[QG系統](http://140.120.13.249:3000)，用來建立Querator_Cam的後端。

## 檔案說明
### Data
- test.py : 主要的程式
- OCR.py : 使用OCR的程式
- ckip.py : 使用中文NER的程式(https://github.com/Cai-Pei-Xuan/Use-from-CkipTagger)
- NER.py : 使用英文NER的程式
- requestment.txt : 紀錄需要安裝的環境
## 環境需求
- python 3.6+
- spacy 2.2.1
- Flask 1.1.1
- ckiptagger: 0.0.19

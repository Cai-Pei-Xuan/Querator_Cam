要使用ocr.space的話provider值給1
{
    "photo":"data:image/....",
    "provider" : "1"
}
回傳的結果
return 
{
    "context": "韓國瑜.....",
    "result": "1"
}

OR
return 
{
    "context": "ocr.space給的錯誤訊息",
    "result": "0"
}

要使用google的OCR的話provider值給除1以外的值
{
    "photo":"data:image/....",
    "provider" : "0"
}
回傳的結果
return 
{
    "context": "韓國瑜.....",
    "result": "1"
}

OR
return 
{
    "context": "error",
    "result": "0"
}
